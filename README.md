# JSON Blog

Simple blogging app using React.

## Getting Started  

1. Clone the repo
2. `npm install`
3. `npm start`              
4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
5. The page will reload when you make changes.
6. You may also see any lint errors in the console.
